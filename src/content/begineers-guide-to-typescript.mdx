## What is TypeScript?

TypeScript is a superset of JavaScript that adds static typing and other features to the language.
It was created by Microsoft in 2012 and has been gaining popularity ever since.
TypeScript is a great language for beginners because it provides a lot of the benefits of static typing without the
complexity of a full-blown statically typed language like Java or C#.
It fixes many of the limitations of JavaScript while keeping all the benefits.
As a result, many employers are interested in developers who can switch between both JavaScript and TypeScript.
TypeScript can help catch errors early in the development process, improving code reliability and reducing the likelihood of runtime errors

## Why TypeScript?

1. Easy to learn: TypeScript is a superset of JavaScript, so it's easy to learn if you already know JavaScript.
2. Improved code quality: TypeScript increases productivity by providing static type-checking, making it easier to detect
and fix errors at an early stage, predicting potential issues, and improving code quality. This can help beginners write better code and avoid common errors.
